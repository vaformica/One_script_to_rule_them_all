# Changelog

## 0.1.0 - Initial MVP

### Added

- Three-tab Streamlit interface.
- Firebird SSH connectivity checks.
- Recursive remote scanning for MP4, AVI, and TOML files.
- TOML metadata inspection through a remote helper script.
- Stable video-TOML analysis-unit IDs.
- Duplicate suppression with SQLite uniqueness constraints.
- Local catalog and run-history database.
- Immutable run creation.
- Run-specific TOML snapshots.
- Safe threshold editing for `area_ths` and `intensity_ths`.
- IDtracker-only, post-processing-only, and sequential execution modes.
- Generated SLURM scripts.
- Remote job submission and monitoring.
- Quick-result retrieval.
- Track PNG QC decisions and notes.
- Accepted-result CSV export.
- HTML architecture and recovery guide.
