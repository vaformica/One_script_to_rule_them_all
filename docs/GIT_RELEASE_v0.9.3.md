# v0.9.3 — Visible track identity and sortable run date

- Burns a four-line banner directly into every BA and fight track image.
- Banner includes camera, cell, analysis, run number, run date, video filename, and record ID.
- Fight PDFs are built from these labeled images, so every PDF page remains identifiable.
- Adds `date_run` to the QC index and a sortable Date run column in the QC GUI.
- Migrates older QC indexes without losing decisions.
