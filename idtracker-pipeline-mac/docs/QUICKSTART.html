IDtracker Mac/SSH Controller

Install

bash scripts/install_mac.sh
bash scripts/launch_mac.sh

Configure

  1. Test the SSH alias and private key.
  2. Set the Firebird recursive search root.
  3. Set the remote run root.
  4. Enter the exact paths to the proven SLURM scripts.
  5. Verify the script paths.

Use

  1. Scan Firebird recursively.
  2. Review matched and ambiguous rows.
  3. Edit TOML thresholds only when necessary.
  4. Submit one test row.
  5. Review the exact sbatch command and job state.
The existing Firebird scripts must use the exported PIPELINE_* variables or search the run-specific input directory. Test one TOML before a large batch.
